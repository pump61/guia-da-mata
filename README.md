# 🌳 Guia da Mata

Aplicativo profissional para identificação botânica, dendrológica, entomológica e diagnóstico do solo. Desenvolvido para estudantes e profissionais da Engenharia Florestal.

---

## 📱 Funcionalidades

### 🌿 Chaveamento Botânico
4 chaves dicotômicas (A, B, C, D) para identificação de angiospermas e gimnospermas nativas e cultivadas do Brasil.

### 🌲 Chaveamento Dendrológico
Identificação vegetativa de espécies arbóreas.

### 🐞 Chaveamento Entomológico
Identificação de 31 ordens de insetos.

### 🧪 Diagnóstico do Solo
- Entrada completa da análise do solo (pH, P, K, Ca, Mg, Al, V%, T)
- Cálculo da necessidade de calagem
- Recomendação de gessagem
- Adubação por hectare e por cova
- Sugestão de fertilizantes comerciais
- Compartilhamento do relatório

### 🎨 Personalização
7 temas disponíveis: Moss, Stone, Mist, Clay, Dusk, Sakura, Ink.
Ajuste de tamanho da fonte.

---

## 🛠️ Tecnologias

- React Native / Expo
- React Navigation
- AsyncStorage

---

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/pump61/guia-da-mata.git
cd guia-da-mata

# Instale as dependências
npm install

# Execute
npx expo start
```

Escaneie o QR code com o Expo Go (Android) ou a câmera (iOS).

---

## 📦 Build para produção

```bash
eas build -p android --profile production
```
